# Bank-Network
Bnak network
Bank Network Simulation using Cisco Packet Tracer

This project demonstrates the design and implementation of a secure and scalable banking network infrastructure using Cisco Packet Tracer. The network simulates real-world banking operations by connecting multiple branches, departments, and users through routers, switches, and end devices.

The project includes configuration of IP addressing, VLANs, routing protocols, and network security features to ensure efficient communication and data protection. Advanced concepts such as IPv6 configuration, SLAAC, DHCP, and access control lists (ACLs) are also implemented to enhance network performance and security.

Key features of the project include:

🏦 Multi-branch bank network topology
🔌 Router and switch configuration
🌐 IPv4 and IPv6 addressing
🔁 Dynamic routing (e.g., RIP/OSPF/EIGRP)
🔐 Network security using ACLs
📡 VLAN segmentation for different departments
⚙️ DHCP server configuration
🔍 Troubleshooting and testing connectivity

This project reflects practical knowledge of network design, configuration, and troubleshooting, making it useful for students and professionals preparing for networking roles or certifications like CCNA.
